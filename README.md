## 使用流程
### 学生使用建议
- 第一次使用，克隆本仓库

```
git clone https://github.com/QiangZiBro/DS2019_NBU
```
- 将项目文件夹(比如`calc`)复制到您的电脑一个位置
- 用您的编辑器（如`pycharm`)打开这个项目(比如`calc`)进行自己的学习和开发
- 将作业代码发到助教邮箱
- 有新的实验项目，从远程更新
```
git pull origin master
```
